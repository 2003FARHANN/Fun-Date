# 💌 Interactive Date Proposal Website

A playful and responsive date-proposal website built with Python and Flask. It presents a fun proposal experience through multiple interactive screens.

## ✨ Features

- Evasive `No` button that moves on mouse and mobile touch
- Animated `Yes` confirmation with confetti
- Date and time selection
- Interactive food selection
- Personalized confirmation message
- Random or manually configurable joke fee
- Final date summary
- SQLite response storage
- Responsive mobile and desktop design
- Docker and Render deployment support
- Automated tests with GitHub Actions

## 💖 Website Flow

1. **Proposal Page**  
   The visitor can select `Yes`, while the `No` button keeps escaping.

2. **Surprise Page**  
   Displays: “WAIT, YOU ACTUALLY SAID YES??”

3. **Date Selection**  
   The visitor chooses a preferred date and time.

4. **Food Selection**  
   The visitor selects a food option.

5. **Confirmation Page**  
   Shows the selected date and time with a playful message.

6. **One Small Fee**  
   Displays a fictional random or fixed fee.

7. **Final Page**  
   Shows the complete date summary with confetti.

> The fee is only a joke. This website does not collect real payment or card information.
